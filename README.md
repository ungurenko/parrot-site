Лендинг для приложения **Parrot** - macOS-приложения для транскрибации аудио, видео и YouTube-роликов.

## 📦 Назначение

- Промо-страница приложения Parrot
- Демонстрирует фичи транскрайбера и путь пользователя к покупке/установке
